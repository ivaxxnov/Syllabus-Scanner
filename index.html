What Can It Do For Me?

Syllabus-Scanner is a student-built tool that employs ChatGPT to analyze syllabus documents to extract essential information like deadlines and weights! It then compiles it into a spreadsheet with many useful functions and an importable calendar file that you can use anywhere!
